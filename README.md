loopback-socket
=========

Loopback module for create Socket.io connections and allow call methods and make subscriptions

## Installation

  `npm install loopback-socket`

## Usage

    var loopbackSocket = require('loopback-socket');

  Output should be `35,666`


## Tests

  `npm test`
